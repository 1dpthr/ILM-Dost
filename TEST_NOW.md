## ❌ If Something's Wrong

### Blank/White Screen
**Fix:**
1. Open browser console (F12)
2. Look for errors (red text)
3. Copy error message
4. Hard refresh (Ctrl+Shift+R)

### Login Screen Not Appearing
**Fix:**
1. Hard refresh (Ctrl+Shift+R)
2. Clear browser cache
3. Try different browser

### "Try Demo Account" Not Working
**Fix:**
1. Check browser console for errors
2. Make sure localStorage is enabled
3. Not in incognito/private mode

### Charts Not Showing
**Fix:**
1. Wait 2-3 seconds after page load
2. Charts take a moment to render
3. Hard refresh if still not showing

### Can't Navigate Between Pages
**Fix:**
1. Check sidebar is visible
2. Click sidebar items directly
3. Hard refresh if router not working
