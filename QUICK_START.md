# Ilm Dost - Quick Start Guide

Welcome to **Ilm Dost** (Your Trusted Learning Companion)! 🎓

## 🚀 Getting Started

### Step 1: Deploy Backend (CRITICAL)

**⚠️ You MUST do this first for authentication to work!**

1. Go to **Make Settings Page**
2. Find the **Supabase** section
3. Click **"Deploy Supabase Function"**
4. Wait for deployment to complete (30-60 seconds)
5. You'll see a success message when done

### Step 2: Create Your Account

1. The app will show a **Sign Up** screen
2. Enter your:
   - **Full Name** (e.g., "Ahmed Khan")
   - **Email** (e.g., "ahmed@example.com")
   - **Password** (minimum 6 characters)
3. Click **"Sign Up"**
4. You'll be automatically logged in!

### Step 3: Start Learning

Once logged in, explore:

📊 **Dashboard** - Your learning overview
🤖 **AI Chat** - Ask questions, get help
📚 **Courses** - Manage your subjects
📝 **Tests** - Track performance
📅 **Schedule** - Plan your time
🎤 **Voice** - Hands-free assistance

## 🔧 Troubleshooting

### "Sign Up Not Working"
✅ **Solution**: Make sure you deployed the backend first (Step 1)

### "Data Not Saving"
✅ **Solution**: 
1. Check backend is deployed
2. Hard refresh (Ctrl+Shift+R)
3. Try logging out and back in

### "Authentication Error"
✅ **Solution**:
1. Clear browser cache
2. Redeploy backend from settings
3. Wait 1-2 minutes for propagation

## 📱 Using the Platform

### Managing Courses
1. Go to **Courses** tab
2. Your courses load automatically (seeded on first visit)
3. Click a course to expand chapters
4. ✅ Check boxes to mark chapters complete
5. Click "Add note" to save study notes
6. Progress updates in real-time!

### Taking Tests
1. Go to **Tests & Analytics**
2. View your test history
3. See performance trends
4. Get AI insights on strengths/weaknesses

### AI Chat Assistant
1. Click **AI Chat** in sidebar
2. Type questions or use quick prompts
3. Get instant, intelligent responses
4. All conversations saved automatically

### Voice Assistant
1. Open **Voice Assistant**
2. Click the microphone button
3. Speak your question clearly
4. Get spoken response
5. View command history

### Schedule Management
1. Go to **Schedule**
2. Click dates on calendar
3. View events for selected date
4. See upcoming deadlines
