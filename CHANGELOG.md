# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2026-05-01

### Added
- Initial release of Ilm Dost
- Privacy-first AI educational platform
- Complete offline functionality with localStorage
- Demo account for instant access (no setup required)
- Smart dashboard with real-time academic performance tracking
- AI Study Assistant with intelligent tutoring support
- Course and subject management system
- Advanced test and analytics features
- Interactive scheduling and calendar
- Voice assistant for hands-free interaction
- Role-based access (Student, Teacher, Admin)
- Responsive design for all devices
- Professional emerald/teal theme
- Local AI integration support (Ollama, LM Studio, LocalAI)
- Optional Supabase backend for cloud sync
- Comprehensive documentation and guides

### Features
- **Dashboard**: Real-time stats, study hours tracking, performance insights
- **AI Chat**: Context-aware conversations, personalized recommendations
- **Courses**: Chapter-level progress, study notes, progress tracking
- **Tests**: Performance analytics, trend analysis, skills visualization
- **Schedule**: Calendar view, event management, deadline tracking
- **Voice**: Speech-to-text and text-to-speech support

### Technical
- React 18.3.1 with TypeScript
- Tailwind CSS v4 for styling
- Vite for build tooling
- Recharts for data visualization
- Lucide React for icons
- Supabase client for optional backend
- LocalStorage for data persistence

### Documentation
- Comprehensive README with quick start guide
- Local AI setup instructions
- Supabase integration guide
- Deployment checklist
- Testing guides
- Contributing guidelines

### Security
- 100% local data storage option
- No external server calls in demo mode
- GDPR compliant by design
- No tracking or analytics
- Open source and transparent
